# MoodTrails

MoodTrails is an engaging web app that helps users track their emotional wellbeing through an interactive visual journey. Users create daily mood entries that form a beautiful, flowing trail of colored particles representing their emotional state. The more consistent users are with logging their moods, the more mesmerizing their trail becomes.

Key Features:
- Interactive particle-based mood visualization
- Daily mood check-ins with emoji selection
- Streak tracking and rewards
- Shareable mood trails
- Mobile-responsive design

Tech Stack:
- HTML5 Canvas for particle animations
- Vanilla JavaScript
- Local Storage for data persistence
- CSS animations
